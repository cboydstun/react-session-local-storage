## MERN Note Taker Application

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:cboydstun/react-session-local-storage.git
   ```
2. Install and Execute React Client Packages

   ```sh
   yarn install
   ```

   ```sh
   yarn start
   ```
